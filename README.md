# The DiaGame Study

The DiaGame Study has collected data on interstitial glucose levels, heart rate, acceleration, step count, dietary intake (macros and micros), bouts of physical activity, insulin administration, and mood over 14 day period in 60 people with diabetes (8 type 1 and 52 type 2) collected using a continuous glucose monitor (FreeStyle Libre Pro IQ), smartwatch with in-house application, and smarphone application.

# Data Access Right
Access to the data can be requested by submitting a data access request to BME.Seccbioimage@tue.nl.
